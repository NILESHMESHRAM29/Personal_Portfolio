My Story

I am a .NET Backend Developer currently working as a Programmer Analyst at Figment Global Solutions Private Limited, Nagpur (since Jan 2025), where I develop and maintain backend systems for GSTCPC/TaxCPC using ASP.NET Core, C#, and Web API — architecting secure RESTful APIs with structured request/response pipelines, JWT-based authentication, and middleware-driven validation layers for high-volume GST return processing and real-time tax computation workflows.

My work involves Redis caching, third-party API integrations, secure file processing, and MinIO-based object storage. I have also built dynamic Excel generation utilities, async programming modules, and supported IIS/cloud deployments with CI/CD practices.

I completed my Master in Computer Applications (MCA) from G H Raisoni College of Engineering and Management, Nagpur (2023–2025), and hold a BSc in Computer Science & Electronics from Dhote Bandhu Science College, Gondia (2019–2022).

I am passionate about designing efficient, production-ready applications and am actively seeking opportunities as a .NET Developer / Backend Developer / API Developer to contribute to scalable and impactful software systems.
